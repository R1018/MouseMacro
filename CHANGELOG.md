# 更新日志 (CHANGELOG)

本文档记录了 MouseMacro 的所有重要更改。

## [2.0.0] - 2024-03-12

### 新增功能
- 🎨 添加主题切换功能（明亮/暗黑/跟随系统）
- 💾 配置持久化，自动保存用户设置
- ✅ 窗口验证功能，录制前检查窗口是否存在
- 🛑 停止录制和停止回放按钮
- 📋 脚本列表快速选择功能
- 📊 实时状态显示
- 🔄 自动检测系统主题

### 改进
- 🖼️ 窗口列表选择对话框优化，支持搜索
- 🎯 改进按钮样式，添加立体边框效果
- 📏 调整窗口大小，确保所有内容可见
- 🚫 程序启动时自动隐藏控制台窗口
- 📝 详细的 README 文档和故障排除指南

### 修复
- 🔧 修复主题切换时的按钮背景问题
- 🐛 修复配置文件读取错误
- ⚡ 优化日志文件管理，自动清理大文件
- 🛠️ 修复窗口选择对话框的主题适配

### 代码重构
- 🏗️ 重构 GUI 初始化逻辑
- 📦 模块化主题管理
- 🔍 添加错误处理和调试支持

## [1.0.0] - 2024-03-10

### 初始发布
- 🎯 基础录制功能（鼠标移动、点击、拖动）
- 🔄 回放功能，支持多次重复
- 🖥️ GUI 界面
- 💾 JSON 格式保存录制脚本
- 📁 Scripts 目录管理

## 版本说明

### 版本号格式
遵循 [语义化版本 2.0.0](https://semver.org/lang/zh-CN/)
- 主版本号：不兼容的 API 修改
- 次版本号：向下兼容的功能性新增
- 修订号：向下兼容的问题修正

### 更新类型
- `新增` - 新功能
- `改进` - 现有功能的改进
- `修复` - 问题修复
- `移除` - 功能移除
- `安全` - 安全性修复
